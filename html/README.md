# Manuscript Project

Простой веб-интерфейс для записи идей в блокчейн Solana.

## Структура
- `index.html` - Главная страница
- `styles.css` - Стили
- `script.js` - Логика работы (Кошелек + Частицы + Блокчейн)

## Как запустить
1. Откройте `index.html` в любом современном браузере.
2. Подключите кошелек Phantom (сеть Devnet).
3. Пишите идеи и сохраняйте их вечно.

## Требования
- Браузер с расширением Phantom Wallet.
- Немного SOL в сети Devnet для оплаты комиссии.
